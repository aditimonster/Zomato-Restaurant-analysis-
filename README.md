# Zomato Restaurant Analysis 🍽️

Analysed 9,000+ Zomato restaurants using Excel.

## Key Findings
- Dining dominates with 20,363 votes
- 83% prefer restaurants with online ordering
- ₹600 for two is India's sweet spot

## Tools Used
- Microsoft Excel (Pivot Tables, Charts)
